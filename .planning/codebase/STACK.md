# Tech Stack

## Backend
- **Runtime**: Node.js
- **Framework**: Express.js (v5.x)
- **Real-time Communication**: Socket.io (v4.x)
- **Media Server (SFU)**: Mediasoup (v3.x)
- **Database**: PostgreSQL (`pg` library)
- **Caching/State**: Redis
- **Security**: JWT (`jsonwebtoken`), Bcrypt (`bcrypt`, `bcryptjs`)
- **Utilities**: `nanoid`, `uuid`, `dotenv`

## Frontend
- **Framework**: React (Vite)
- **Styling**: TailwindCSS
- **Structure**: Component-based architecture
- **Logic**: React Hooks & Context API
- **Communication**: Socket.io Client
- **WebRTC**: Mediasoup-client

## Dev Tools
- **Runner**: Nodemon
