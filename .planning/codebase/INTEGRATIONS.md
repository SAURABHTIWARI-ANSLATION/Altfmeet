# External Integrations

## Databases
- **PostgreSQL**: Used for persistent storage (users, meeting history, etc.).
- **Redis**: Likely used for real-time signaling state, room management, or caching.

## Media Handling
- **Mediasoup**: Self-hosted SFU (Selective Forwarding Unit) for WebRTC media streams.

## Security
- **JWT**: For stateless authentication.
- **Bcrypt**: For password hashing.
